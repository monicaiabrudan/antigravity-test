# Project 3 Solution: Identifying Host Genetic Factors for Resistance to SARS-CoV

In this project, we characterize genetic variants within the *Trim55* gene (chromosome 3) across 52 inbred mouse strains using a VCF file derived from the Mouse Genomes Project (MGP). The goal is to identify candidate mutations that could drive susceptibility or resistance to SARS-CoV infection, specifically targeting the **HrS1** locus (which regulates perivascular cuffing and lung inflammation).

---

## 🛠️ Data Retrieval & Environment Setup

First, we created the project directory and downloaded the VCF file for Chromosome 3 (region `18,000,000` to `22,000,000` bp):

```bash
# Create project directory
mkdir -p $HOME/Project3 && cd $HOME/Project3

# Download the VCF directly from Google Drive
wget -O "mgp_REL2005_snps_indels.3.18000000-22000000.vcf.gz" \
  "https://drive.google.com/uc?export=download&id=1dNfcO49nov0MzA2UIuOStJIlc1y_oZ3w"
```

Next, we index the downloaded VCF file to enable quick region querying:

```bash
bcftools index mgp_REL2005_snps_indels.3.18000000-22000000.vcf.gz
```

## 📋 Task 1: Sample Verification and Trim55 Extraction

### 1. Confirming the number of samples (strains)
We query the list of samples from the VCF header and count them:

```bash
bcftools query -l mgp_REL2005_snps_indels.3.18000000-22000000.vcf.gz | wc -l
```
> [!NOTE]
> **Output:** `52`
>
> There are exactly **52 samples** (inbred mouse strains) in the VCF file.

### 2. Identifying Trim55 coordinates and extracting variants
The coordinates of *Trim55* (GRCm38/mm10) are approximately `3:19639519-19697415` (covering all variants in the VCF annotated with `SYMBOL=Trim55` in the VCF's CSQ info).

We extract the variants in this region and save them to a new VCF file:

```bash
bcftools view -r 3:19639519-19697415 \
  mgp_REL2005_snps_indels.3.18000000-22000000.vcf.gz \
  -O z -o Trim55.vcf.gz

# Index the extracted file
bcftools index Trim55.vcf.gz
```

Counting the extracted variants:
```bash
bcftools view -H Trim55.vcf.gz | wc -l
```
> [!NOTE]
> **Output:** `2711`
>
> There are exactly **2,711 variants** located in the *Trim55* gene region.

## 🔍 Task 2: Characterizing Functional Variants

To understand how variants affect gene functions, we inspect specific positions using the `CSQ` (Consequence) field. The format of `CSQ` in this VCF is:
`Allele|Consequence|IMPACT|SYMBOL|Gene|Feature_type|Feature|BIOTYPE|...|Protein_position|Amino_acids|Codons|...|SIFT`

### 1. Querying Position `3:19671000`
```bash
bcftools query -f '%CHROM\t%POS\t%REF\t%ALT\t%INFO/CSQ\n' -r 3:19671000 Trim55.vcf.gz
```

* **Genomic Variant:** `C` to `T` (SNV)
* **Consequence:** `missense_variant` (missense mutation) with `MODERATE` impact on `Trim55`.
* **Codon Change:** `aCt` to `aTt` (amino acid change: **T227I**, Threonine to Isoleucine at position 227).
* **SIFT Prediction:** `tolerated(0.08)` (near the `0.05` deleterious threshold).

## 🧬 Task 4: Identifying Missense Variants in Trim55

Most of the 2,711 variants are harmless. We filter exclusively for **missense variants in Trim55** and write them to a text file. There are **11 distinct genomic locations (sites)** corresponding to **12 alleles** (the site `3:19672870` is multiallelic and contains two distinct missense alleles: `G/C` and `G/T`).

We extract and format these into `Trim55_missense_variants.txt`:

```bash
bcftools query -f '%CHROM\t%POS\t%REF\t%ALT\t%INFO/CSQ\n' Trim55.vcf.gz | \
  awk -F'\t' 'BEGIN {print "CHROM\tPOS\tREF\tALT\tCONSEQUENCE\tIMPACT\tSYMBOL\tPROTEIN_POS\tAMINO_ACIDS\tCODONS\tSIFT"} {
    n = split($5, annotations, ",");
    for (i = 1; i <= n; i++) {
      split(annotations[i], fields, "|");
      if (fields[2] == "missense_variant" && fields[4] == "Trim55") {
        print $1 "\t" $2 "\t" $3 "\t" $4 "\t" fields[2] "\t" fields[3] "\t" fields[4] "\t" fields[15] "\t" fields[16] "\t" fields[17] "\t" fields[25];
      }
    }
  }' > Trim55_missense_variants.txt
```

## 💡 Biological Discussion & Interpretation

### 1. Classical vs. Wild-derived Genetic Architecture
A key finding is that these missense mutations are absent from all standard classical laboratory inbred strains (e.g., `A/J`, `NOD_ShiLtJ`, `C57BL/6NJ`, `BALB/cJ`, `DBA/2J`). Instead, they are found almost exclusively in **wild-derived inbred strains** representing different evolutionary branches of the genus *Mus*:
* **CAST_EiJ** (*M. m. castaneus*)
* **MOLF_EiJ** (*M. m. molossinus*)
* **SPRET_EiJ** (*Mus spretus*)
* **CZECHII_EiJ** (*M. m. musculus*)
* **JF1_MsJ** (*M. m. molossinus/domesticus*)

### 2. Identifying Candidate Susceptibility/Resistance Drivers
In the PLoS Genetics study, **CAST/EiJ** was identified as extremely susceptible to SARS-CoV pathogenesis (reaching clinical endpoints within 4 days).
Integrative statistical mapping in Collaborative Cross (CC) strains identified **Trim55** within the **HrS1** QTL, which regulates **perivascular cuffing (inflammation around blood vessels)** in the lungs:
* **CAST/EiJ** carries **6 missense mutations** in *Trim55* (G220S, G343V, E346D, A354T, S367T, and A384T).
* Strains inheriting the **CAST/EiJ haplotype** at the *HrS1* locus display a **reduction in perivascular cuffing and lung inflammation** following infection.
* Interestingly, **Trim55 knockout mice** (*Trim55⁻/⁻*) on a C57BL/6J background **phenocopy this CAST/EiJ trait**, exhibiting a significant reduction in vascular cuffing and monocyte recruitment.
* **Conclusion:** This indicates that the CAST/EiJ allele of *Trim55* (carrying these 6 missense mutations) likely acts as a **natural hypomorph or loss-of-function allele**. It impairs the function of the *Trim55* E3 ubiquitin ligase, reducing the recruitment of inflammatory cells and thus ameliorating tissue damage (vascular cuffing) in the lungs during SARS-CoV infection.

### 3. The 3:19672870 Position
This position is particularly interesting because it is **multiallelic** and differs across wild-derived lineages:
* **CAST_EiJ** and **SPRET_EiJ** have the **C** allele (**S367T** mutation).
* **MOLF_EiJ** has the **T** allele (**S367I** mutation).
This indicates that the Serine at position 367 is highly polymorphic in wild-derived strains, further highlighting it as a hotspot for functional divergence.
